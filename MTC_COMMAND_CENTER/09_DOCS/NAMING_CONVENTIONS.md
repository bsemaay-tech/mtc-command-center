# Naming Conventions

Use short stable IDs:

- Task: `MCC-BOOT-002`
- Report: `RPT-2026-0001`
- User input: `USR-IN-2026-0001`
- TradingView export: `TW-CASE-001-20260530-001`
- QuantLens candidate: `QLR-2026-0001`

Avoid full video titles or long strategy names in folder names. Put descriptive titles inside manifests and reports.
