# Gemini Task Template: Intake Review Task

## Purpose

Review user-provided transcripts, notes, or strategy documents and decide whether they are ready for QuantLens intake.

## Checklist

- Source is identified.
- Entry logic is described.
- Exit logic is described.
- Risk model is described or marked missing.
- Required data is known.
- Testable hypotheses are extracted.

## Output

Create an intake review report and recommend `ACCEPT_FOR_RESEARCH`, `NEEDS_CLARIFICATION`, or `REJECT_FOR_NOW`.
