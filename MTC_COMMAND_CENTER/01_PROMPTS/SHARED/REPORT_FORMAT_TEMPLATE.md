# Report Format Template

## Summary

Short result summary and task outcome.

## Inputs Reviewed

List source files, exports, reports, transcripts, or commands reviewed.

## Work Performed

Describe actions taken without hiding failures.

## Findings

List evidence-backed findings. Do not invent missing metrics.

## Status Updates

List status files updated and fields changed.

## Blockers

Mark missing source data as `WAITING_FOR_USER`.

## Next Actions

Recommend concrete next tasks.
