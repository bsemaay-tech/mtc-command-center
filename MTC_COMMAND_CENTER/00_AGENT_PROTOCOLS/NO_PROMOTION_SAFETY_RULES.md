# No-Promotion Safety Rules

These rules are permanent safety constraints for MTC Command Center handoffs.

- A research-only artifact is not paper-ready.
- Gate 3 incomplete blocks promotion.
- `promotable=false` must be preserved.
- `SOURCE_NAKED` is not `MTC_LIGHT`.
- `SOURCE_NAKED` is not `RISK_NORMALIZED`.
- Crypto/USDT evidence is not native US equities evidence.
- No top-results ranking without an approved same-profile result set.
- No Pine, MTC_V2, backtest engine, live, or broker changes without explicit user approval.

## Current GEN RSI Checkpoint

The accepted GEN RSI checkpoint is source-naked research only. It remains `SOURCE_NAKED`, `RESEARCH_ONLY`, `promotable=false`, with Gate 3 incomplete. Do not create new artifacts or `top_results.json` from that checkpoint without explicit approval.
