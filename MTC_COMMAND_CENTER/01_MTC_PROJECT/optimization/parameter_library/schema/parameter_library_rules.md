# Parameter Library Rules

Research seeds only; not Pine defaults; not production parameters.

- Store research seeds by producer namespace.
- Keep producer seeds separate from exit, risk, filter, and regime-mitigation seeds.
- Every seed must link to source run/report evidence.
- Every seed must preserve dataset/source/hash context when available.
- Medium candidates may seed second-pass refinement only.
- Strict robust candidates are required before promotion packaging.
- No library entry may modify Pine defaults by itself.
