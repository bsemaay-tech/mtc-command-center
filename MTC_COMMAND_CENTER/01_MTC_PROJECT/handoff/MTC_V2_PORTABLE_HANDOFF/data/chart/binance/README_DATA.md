# Binance Chart Data

This folder contains approved chart CSV data for local MTC V2 development, parity smoke work, and future optimization planning.

Current included dataset:

- Symbol: BTCUSDT.P
- Exchange: BINANCE
- Timeframe: 1h
- Source type: tradingview_chart_csv_binance
- File: data/chart/binance/BTCUSDTP/1h/BINANCE_BTCUSDT.P, 60_consolidated_stable.csv

TradingView Strategy Tester XLSX exports are not chart CSV data and are not included in this main package.
