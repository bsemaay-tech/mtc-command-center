# PyneCore Oracle POC

Purpose: test PyneCore as an experimental, free local strategy-execution oracle for small manually translated MTC modules.

Rules:
- Do not use PyneComp paid compiler.
- Do not treat PyneCore as a TradingView replacement.
- Translate only small modules manually or with AI.
- Promote only after parity reports pass against the Python engine and TradingView exports where available.

Run:

```powershell
.\run_pynecore_poc.ps1
```

Outputs are written under `output/` and install errors under `reports/`.
