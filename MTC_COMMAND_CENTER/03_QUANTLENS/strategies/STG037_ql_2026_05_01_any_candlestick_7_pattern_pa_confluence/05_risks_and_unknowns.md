# Risks And Unknowns

## Main Risks
- Pattern definitions can vary across traders and platforms.
- Support/resistance and trendlines can create lookahead bias.
- "Next resistance/support" target is undefined without a level algorithm.
- Broad pattern library can overfit if all patterns are tested together.
- Candlestick patterns alone have weak edge without context.

## Unknowns
- Exact source video URL.
- Channel name and channel ID.
- Timeframe and market used in examples.
- Whether entries require close confirmation or stop orders.
- Whether targets are fixed R or next level.

## Mitigations
- Treat patterns as confirmation, not standalone signals.
- Use one narrow prototype first.
- Use past-only level construction.
- Compare each pattern separately before combining.
