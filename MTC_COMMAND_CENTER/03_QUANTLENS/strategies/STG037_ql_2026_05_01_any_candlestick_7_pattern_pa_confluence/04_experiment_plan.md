# Experiment Plan

## Prototype Scope
Do not prototype all seven patterns at once. Start with one pattern family and one confluence rule.

## Recommended First Variant
`Engulfing pattern + support/resistance confluence`

## Deterministic Rules To Define
- Candle body/wick ratio thresholds for pin bars and doji.
- Engulfing definition: full high/low engulf or body-only engulf.
- Morning/Evening Star size thresholds.
- Required prior move length before reversal patterns.
- Support/resistance construction using past-only pivots or rolling levels.
- Level proximity tolerance.
- Entry execution: stop order above/below pattern or close confirmation.
- Stop anchor and buffer.
- Target: next level or fixed R.

## Explicit Non-Goals
- No seven-pattern combined system initially.
- No discretionary trendline drawing.
- No future-pivot support/resistance.
- No Pine patch.
- No production runner change.
- No optimization before deterministic event extraction.
