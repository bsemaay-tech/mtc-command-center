# 0. Strategy Source Intake

## Amaç

Strateji kaynağından yapılandırılmış bilgi çıkarmak.

---

# Kaynak Tipleri

```text
- YouTube video transcript
- YouTube ekran görüntüsü / görsel anlatım
- TradingView indicator açıklaması
- Pine Script kodu
- Blog / makale
- Twitter / X thread
- PDF / eğitim notu
- Manuel olarak anlatılmış strateji fikri
```

---

# Çıkarılacak Bilgiler

```text
Strategy name:
Source:
Market:
Timeframe:
Long rules:
Short rules:
Exit rules:
Indicators:
Parameters:
Risk model:
Signal timing:
Bar close / intrabar:
Manual interpretation required:
Known repaint risk:
MTC compatibility notes:
```

---

# Intake Değerlendirme Notları

Bu aşama puanlama aşaması değildir. Amaç stratejiyi yapılandırılmış forma dönüştürmektir.

Eksik bilgi varsa strateji doğrudan reddedilmez. Eksikler Gate 1 değerlendirmesinde puanı düşürür.

---

# Intake Çıktısı

```text
- Strategy source summary
- Rule extraction draft
- Indicator list
- Parameter list
- Preliminary ambiguity list
- Preliminary MTC compatibility notes
```
