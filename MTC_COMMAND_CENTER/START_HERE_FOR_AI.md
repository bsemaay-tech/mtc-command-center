# Start Here For AI Workers

Follow this order before doing any work in MTC Command Center:

1. Read `MTC Command Center ARCHITECTURE.md` first.
2. Read `CURRENT_STATUS.md`.
3. Read `AI_OPERATING_RULES.md`.
4. Read `02_TASKS/TASK_QUEUE.json`.
5. Pick only the assigned task.
6. Do not modify MTC_v2 core files.
7. Do not touch PineTS/parity engine unless explicitly assigned.
8. Create report first.
9. Update status files after task.
10. Update `02_TASKS/TASK_HISTORY.json` after task.

If required source data is missing, mark the task as `WAITING_FOR_USER` and explain exactly what the user must provide.
